what is strictmode?

StrictMode is a feature in JavaScript that was introduced in ECMAScript 5 (ES5). It is a way to enforce a stricter set of rules for JavaScript execution, helping developers write more secure and error-free code.

what is Linting Error?

A linting error occurs when a code analysis tool (called a linter) detects issues in your code that do not adhere to predefined coding standards, style guidelines, or best practices. These errors do not typically prevent the code from running, but they highlight areas that could lead to bugs, reduce readability, or violate conventions.
